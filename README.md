# Hand Gesture Laptop Control

Control your **laptop’s mouse, clicks, and volume** using **hand gestures** with Python.

---

## Features
- **Move mouse:** Index finger  
- **Left click:** Two fingers  
- **Right click:** Three fingers  
- **Volume up:** Thumb up  
- **Volume down:** Index finger only  

---

## Requirements
- **Python 3.8+**  
- **Webcam**
